# Компонент битрикс - многоуровневое меню меню

Ищет файл меню в папке `/includes/menus/`

Имя файла должно быть следующего вида: `<название меню>.menu.php`

## Подсключение компонента

```php
$APPLICATION->IncludeComponent(
    'falur:menu',         
    '',         
    array(
        'MENU' => 'top',
    )
);
```

## Файл с меню

```php
return [
    [
        'NAME' => 'Главная',
        'LINK' => '/'
    ]
]
```
